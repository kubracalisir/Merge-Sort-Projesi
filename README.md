# Merge-Sort-Projesi
Merge Sort Project given by Patika.dev

[16,21,11,8,12,22] -> Merge Sort

Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
Big-O gösterimini yazınız.

<p> [16,21,11]           [8,12,22]
<p> [16,21]   [11]         [8,12]   [22]
  <p>  [16]   [21]  [11]       [8]  [12]  [22]
    <p> [16,21]  [11]        [8,12]   [22]
      <p> [11,16,21]    [8,12,22]
        <p> [8,11,12,16,21,22]
          
          <p> Best: O(nlog_2 n)
            <p> Average: O(nlog_2 n)
              <p> Worst: O(nlog_2 n)
